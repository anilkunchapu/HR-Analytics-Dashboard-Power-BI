# HR Analytics Dashboard – Power BI

## Project Overview
This project presents an interactive HR Analytics dashboard built using Power BI to analyze employee attrition and workforce insights.

## Dataset
The dataset contains employee information such as:
- Age
- Department
- Job Role
- Salary
- Job Satisfaction
- Education Field
- Marital Status

## Key Metrics
- Total Employees: 1470
- Active Employees: 1233
- Attrition Count: 237
- Attrition Rate: 16.12%
- Average Age: 37

## Dashboard Insights
- R&D and Sales departments have the highest attrition
- Laboratory Technicians and Sales Executives show higher turnover
- Employees aged 26–35 have higher attrition
- Lower salary bands show more employee turnover

## Tools Used
- Power BI
- DAX
- Data Visualization
- Data Modeling
